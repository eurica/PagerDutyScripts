PagerDuty Scripts

An unofficial list of scripts, code samples and libraries with PagerDuty's API.

Name Description Technology
PDJS Javascript library to access PagerDuty REST and event APIs JavaScript
Email to webhook Simple PHP scripts to send out notifications by email on state changes. PHP
Ruby/HTTParty example Simple example of how to hit PagerDuty's API with Ruby and HTTParty Ruby
Meteor.js Populate a Status board from PD with Meteor node.js
PagerDuty Webscripts A collection of Lua scripts for webscript.io to trigger a conference call, integrate with HipChat or Slack and trigger an incident from a failed heartbeat. Lua
Grappling A tiny, hacky DSL for inflatable Webhooks for Ruby and PagerDuty. It uses Sinatra, Resque, and Bootstrap. Bring your own Redis install, or just get a free one from redis to go. Ruby
On-call API examples Shell script one-liners and a JavaScript bookmarklet node.js
Phoneduty Dispatch incoming telephone voicemails/SMS messages from Twilio according to a PagerDuty on-call schedule. A much improved version of an earlier code sample Python
Hubot integration PagerDuty can now participate in any chatroom that Hubot supports thanks to Github and contributors. JavaScript
PagerDuty Ruby gem A library for triggering and resolving incidents. [rubygems] Ruby
pagerdutyduty Set schedules in PagerDuty via YAML, with the resulting schedule created in PagerDuty via overrides. Python
Pygerduty Python Library for PagerDuty's REST API maintained by Dropbox Python
lita-pagerduty A plugin for the Lita chatbot for checking who's on call, scheduling, ack, resolve, etc. Works with any chatroom supported by Lita Tool
Zoho Reporting Zoho reports can import PagerDuty data and run SQL queries against it. This blog post shows how to generate reports from your incidents. Tool
pagerduty_dashing A Dashing dashboard for PagerDuty services and on-call status Ruby
"Who is on call?" A quick PDJS example of how to load who is on call now and display it in a webpage. JavaScript

About this list

This is a labour of love from some PagerDuty employees -- but none of the scripts here are officially supported by PagerDuty. If you'd like to add something to this page either email dave@pagerduty.com, or even better submit a pull request to https://github.com/eurica/pagerdutyscripts.
